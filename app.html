About

Student Finance Tracker

A money tracker built for students. No server, no signup, just your data.

Purpose

See where your money goes across Food, Books, Transport and more. Check 7-day spending, set a cap and export your data any time.

Features

  • Income, expense & savings tracking
  • Set a starting balance, see it update live
  • 7-day spending chart
  • 3 currencies (USD, EUR, RWF)
  • Fast search across all transactions
  • Spending cap with alerts
  • Import and export as JSON
  • Saves in your browser, no account needed

Keyboard Shortcuts

  • Alt+N - Add transaction
  • Alt+1-5 - Jump between sections
  • ← → - Move between tabs
  • Esc - Close popup
